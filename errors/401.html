Error 401: Unauthorized

Normally caused by:

  • Authentication is required and the user was not signed in.
  • The user signed in but entered incorrect credentials.
  • The user was signed in but enough time has passed where ezClocker expired the current sign in and requires the user to re-sign in.
For more information visit Offical Standard: RFC-9110 HTTP Semantics
